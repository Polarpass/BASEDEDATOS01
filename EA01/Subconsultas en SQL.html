🧠 Subconsultas en SQL

Guía visual usando esquema HR (Oracle)

🔹 Subconsulta de UNA FILA

Devuelve un solo valor. Se usa con =, >, <.
SELECT first_name, salary FROM employees WHERE salary > ( SELECT salary FROM employees WHERE hire_date = ( SELECT MIN(hire_date) FROM employees ) );
💡 Se comporta como un número dentro del WHERE

🔹 Subconsultas MÚLTIPLES FILAS

Devuelven varios valores → usar IN, ANY, ALL
SELECT first_name, department_id FROM employees WHERE department_id IN ( SELECT department_id FROM departments WHERE department_name IN ('IT','Sales') );

🔹 Subconsulta en FROM

Actúa como tabla temporal
SELECT * FROM ( SELECT department_id, AVG(salary) avg FROM employees GROUP BY department_id ) WHERE avg > 8000;

🔹 Subconsulta en SELECT

Calcula valores por cada fila
SELECT department_name, (SELECT COUNT(*) FROM employees e WHERE e.department_id = d.department_id) total FROM departments d;

🔹 Crear tabla con subconsulta

Permite generar nuevas tablas
CREATE TABLE empleados_altos_salarios AS SELECT * FROM employees WHERE salary > ( SELECT AVG(salary) FROM employees );
